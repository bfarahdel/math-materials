Example 2Binomial Theorem

Expanding $(a + b)^7$ using two different methods.

Key Concept: Binomial Theorem

$$(a + b)^n = \sum_{k=0}^{n} {}_nC_k a^{n-k}b^k$$

Replace $n$ with 7 to determine the coefficients and exponents.

Combination Formula

The coefficients are calculated using the combination formula:

$${}_nC_r = \frac{n!}{r!(n - r)!}$$

Where $!$ denotes a factorial (e.g., $3! = 3 \times 2 \times 1 = 6$).

Method 1: Use Combinations

Method 1

The expansion using combination notation:

$$(a + b)^7 = {}_7C_0 a^7 + {}_7C_1 a^6b^1 + {}_7C_2 a^5b^2 + \dots + {}_7C_7 b^7$$

Calculating the coefficients using the formula above:

$$1a^7 + 7a^6b + 21a^5b^2 + 35a^4b^3 + 35a^3b^4 + 21a^2b^5 + 7ab^6 + 1b^7$$
