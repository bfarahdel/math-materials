Problem 24

Identification

Determine if the following is a polynomial:

$$2x^2 - 3x + 5$$
Analysis

The exponents are $2$ and $1$. Since all exponents are non-negative integers and no variables appear in denominators or radicals, it qualifies.

Result

Yes

Degree: $2$ (Quadratic)

Problem 25

Identification

Determine if the following is a polynomial:

$$a^3 - 11$$
Analysis

The variable $a$ is raised to the 3rd power. The constant $-11$ is allowed (equivalent to $-11a^0$).

Result

Yes

Degree: $3$ (Cubic)

Problem 26

Identification

Determine if the following is a polynomial:

$$\frac{5np}{n^2} - \frac{2g}{n}$$
Analysis

This expression contains a variable $n$ in the denominator. Since $\frac{1}{n} = n^{-1}$, and polynomials cannot have negative exponents, it fails.

Result

No

Problem 27

Identification

Determine if the following is a polynomial:

$$\sqrt{m - 7}$$
Analysis

Polynomials cannot have variables under a radical. $\sqrt{m}$ is equivalent to $m^{1/2}$, which is a fractional exponent.

Result

No

Problem 28

Simplification

Simplify the expression:

$$(6a^2 + 5a + 10) - (4a^2 + 6a + 12)$$
Step 1: Distribute Negative

$$6a^2 + 5a + 10 - 4a^2 - 6a - 12$$

Step 2: Group & Combine

$$(6a^2 - 4a^2) + (5a - 6a) + (10 - 12)$$

Final Answer
$$2a^2 - a - 2$$

Problem 29

Simplification

Simplify the expression:

$$(7b^2 + 6b - 7) - (4b^2 - 2)$$
Step 1: Distribute Negative

$$7b^2 + 6b - 7 - 4b^2 + 2$$

Step 2: Group & Combine

$$(7b^2 - 4b^2) + 6b + (-7 + 2)$$

Final Answer
$$3b^2 + 6b - 5$$
