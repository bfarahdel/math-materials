例题 2c

Negative Exponents

负指数问题

Determine whether the expression is a polynomial. If not, explain why.

判断该表达式是否为多项式。如果不是，请说明理由。

Analyzing \(x^{-3}\) 分析 \(x^{-3}\)

The exponent is \(-3\). Polynomials must have whole number exponents (\(0, 1, 2...\)). Note that \(x^{-3} = \frac{1}{x^3}\).

指数为 \(-3\)。多项式的指数必须是整数（\(0, 1, 2...\)）。注意 \(x^{-3} = \frac{1}{x^3}\)。

Analyzing \(2x^{-2}\) 分析 \(2x^{-2}\)

Similarly, the exponent \(-2\) is not a whole number. This is equivalent to \(\frac{2}{x^2}\).

同样地，指数 \(-2\) 不是整数。这相当于 \(\frac{2}{x^2}\)。

Conclusion 结论

Exponents must be positive integers or zero. This expression is not a polynomial.

指数必须是正整数或零。因此该表达式不是多项式。
