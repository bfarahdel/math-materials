Symmetry

对称性

Lesson 3-5 | 第 3-5 课

Introduction引言

Symmetry exists if a rigid motion maps a figure onto itself.

如果一个“刚性运动”将图形映射到自身，则该图形具有对称性。

Rigid motions include / 刚性运动包括：

  • Reflection / 反射
  • Translation / 平移
  • Rotation / 旋转
  • Glide Reflection / 滑移反射

Line Symmetry轴对称

A figure has line symmetry if it maps onto itself by a reflection in a line (line of symmetry).

如果图形通过直线的反射映射到自身，则该图形具有轴对称（该直线称为对称轴）。

Bilateral / 两侧对称

1 Line / 1 条对称轴

Asymmetric / 不对称

0 Lines / 0 条对称轴

Rotational Symmetry旋转对称

The figure maps onto itself by a rotation between 0° and 360° about the center of symmetry.

图形绕对称中心旋转 0° 到 360° 之间的一个角度后与自身重合。

Hexagon / 六边形

• Order / 阶: 6

• Magnitude / 幅度: 60°
