Compound Angle Formulae

Sine Addition/Subtraction

sin(A±B)=sinAcosB±cosAsinB

Cosine Addition/Subtraction

cos(A±B)=cosAcosB∓sinAsinB

Tangent Addition/Subtraction

tan(A±B)= tanA±tanB1∓tanAtanB

Reference
sin(A±B)sinA cosB ± cosA sinB
cos(A±B)cosA cosB ∓ sinA sinB

Solve the equation sin(60° - x) = 2 sin x for 0° < x < 360°

Step 1: Expand using Compound Angle Formula sin 60° cos x - cos 60° sin x = 2 sin x
Step 2: Substitute Exact Values 32cosx - 12sinx =2sinx
Step 3: Eliminate Fractions (Multiply by 2) √3 cos x - sin x = 4 sin x
Step 4: Group like terms √3 cos x = 5 sin x
Step 5: Rearrange to find tan x 35=tanx
Step 6: Solve for x

Basic angle = tan⁻¹(√3/5) ≈ 19.1°

Since tan x is positive, x is in Quadrant I or III:

x = 19.1° or 199.1°

Solution: Question 4

Finding cos(x - 60°)

Step 1: Identify required parts

cos(x - 60°) = cos x cos 60° + sin x sin 60°

We have cos x = 4/5. We need sin x.

Step 2: Find sin x

Since 0° < x < 90°, x is in Quadrant I (all positive).

sin² x = 1 - (4/5)² = 1 - 16/25 = 9/25

sin x = √(9/25) = 3/5

Step 3: Substitute & Calculate = (4/5)(1/2) + (3/5)(√3/2) = 4/10 + (3√3)/10
Step 4: Final Value 4+3310
